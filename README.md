# Emotionally Bulletproof — Landing Page

Landing page for Hridaya Breathwork's flagship 9-week 1:1 program, **Emotionally Bulletproof**.

## What's here

- `Emotionally Bulletproof - Landing Page V1.html` — the full page. Self-contained: paste the whole block into a single Squarespace Code Block, or open the file directly in a browser to preview.
- `photos/` — hero, section photos, the 7 Belts of Tension graphic, and the bottom carousel images.
- `icons/` — brand background icons (hands sunburst, moon rays, flower of life) in blue/gold/deep-blue variants.

## Before publishing

Replace `ACUITY_AUDIT_CALL_URL` (used on every call-to-action button) with the live **Clarity Call** booking link.
